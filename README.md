# Responsive Blog Website – Jane's Fashion Blog
## Project Overview

This project is a fully responsive fashion blog website built using HTML, CSS, and Font Awesome. It demonstrates modern responsive design techniques, including flexible layouts, media queries, and adaptive UI components. The website is designed to provide a clean, stylish, and user-friendly experience across devices, from desktop to mobile.

Project Name: Responsive-Blog
Role: Full Front-End Developer (focus on responsive design and UI)

## Features

- **Responsive Design:** Website layout adjusts seamlessly to desktop, tablet, and mobile screens using CSS media queries.
- **Navigation Bar:** Fully styled navbar with links, search bar, and a collapsible hamburger menu for mobile devices.
- **Hero Header:** Eye-catching hero section with background image and a subscription call-to-action.
- **Blog Posts Section:** Blog post cards with images, titles, descriptions, and dates.
- **Social Media Integration:** Multiple social media sections with clickable icons linking to Facebook, X (Twitter), Instagram, and LinkedIn.
- **Footer:** Includes “Back to Top” functionality and attribution.
- **Typography & Icons:** Clean typography using Google Fonts and Font Awesome icons for enhanced UI.

## Technologies Used

- HTML5 – Semantic structure and content organization.
- CSS3 – Styling, layout, and responsive design.
- Font Awesome – Social media icons and other UI elements.
- Google Fonts (Oswald) – Custom typography.

## Responsive Behavior

- Desktop: Full navbar with links and search bar, hero image visible, blog cards in row layout.
- Tablet: Flexible wrapping of blog cards, social media section adapts.
- Mobile: Hamburger menu appears, hero section hides, blog posts stacked vertically, simplified layout for readability.
